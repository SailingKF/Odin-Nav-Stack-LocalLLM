# Odin Nav Stack Local LLM Docs

This repository stores the foundation documentation for a location-triggered tour guide application layer built on top of Odin Nav Stack.

## Highlights

- Map point / POI triggered explanations and guided narration
- Local LLM based explanation and follow-up Q&A pipeline
- Android-friendly control and debug UI integration
- Deployment path designed for embedded platforms such as Jetson Orin NX
